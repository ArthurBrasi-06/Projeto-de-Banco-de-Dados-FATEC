## Backend do Projeto

Esta pasta contém os arquivos responsáveis pela lógica do sistema e conexão com o banco de dados.

###  Funcionalidades
- CRUD de usuários
- CRUD de eventos
- CRUD de páginas do evento
- CRUD de questionários
- Arquivo de conexão com o banco
- Páginas PHP para testar e executar operações

###  Tecnologias
- PHP
- MySQL
- phpMyAdmin
- Servidor local (XAMPP, WAMP ou similar)
